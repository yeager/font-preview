# Font Preview

Font browser with Unicode coverage analysis per language.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install font-preview
```

### Fedora/RPM
```bash
sudo dnf install font-preview
```
